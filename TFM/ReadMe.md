## TFM – Integración de Datos para Trazabilidad y Análisis en la Industria Médica

Descripción general

Este repositorio contiene un notebook en Python desarrollado como parte del Trabajo Fin de Máster (TFM) del Máster en Big Data & Business Intelligence.
El objetivo principal del notebook es demostrar una metodología de análisis exploratorio de datos (EDA) y modelado de información orientado a grafos, aplicada a un caso de uso industrial de trazabilidad en la industria médica.
El trabajo se centra en la integración de datos provenientes de sistemas productivos, su estructuración lógica y su posterior explotación analítica mediante tecnologías de grafos y visualización.

## Anonimización y confidencialidad

Aviso importante: Este notebook utiliza datos sintéticos, nombres genéricos y ejemplos simulados. No contiene información real, sensible ni propietaria de ninguna empresa, producto, línea de producción, proceso, equipo u operario. Su único propósito es académico, demostrando la arquitectura, la lógica de análisis y el enfoque metodológico empleado en el TFM.

Objetivos del notebook

- Aplicar Análisis Exploratorio de Datos (EDA) sobre datos industriales simulados.
- Diseñar un modelo de datos orientado a grafos para representar relaciones de trazabilidad.
- Mostrar cómo los datos pueden integrarse y estructurarse para: Análisis de defectos, relación entre procesos y resultados, soporte a decisiones basadas en datos y servir como base para la visualización posterior en herramientas de Business Intelligence.
